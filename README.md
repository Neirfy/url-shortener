# Микросервис для сокращения ссылок

- `POST /shorten` — принимает в body оригинальную ссылку и возвращает короткий ID.
- `GET /{short_id}` — редиректит на оригинальную ссылку по короткому ID.
- `GET /stats/{short_id}` — возвращает количество редиректов для короткой ссылки.

## Запуск в Docker

    make docker

## Ссылка на swagger

    http://127.0.0.1:8000/api/v1/docs#/

## Tests

    make test
